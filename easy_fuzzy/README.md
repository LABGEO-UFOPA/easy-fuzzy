# Easy Fuzzy

A QGIS plugin with a step-by-step interface for complete fuzzy modeling workflows.

## Current version
Version 1.1.0

## Main features
- Step-by-step guided workflow
- Input raster management
- Fuzzy membership configuration
- Fuzzy overlay execution
- Sensitivity analysis
- Reclassification using equal intervals or Jenks natural breaks
- Validation using an input validation layer
- HTML validation report export
- Progress bar during processing

## Modules
- Project
- Input Rasters
- Fuzzy Configuration
- Overlay
- Results
- Sensitivity Analysis
- Reclassification
- Validation
- About
